## インターフェイス

## default

インターフェイス内に記述できるメソッドは抽象メソッドのみであり、具体的な実装を記述することができない。

```Java
interface A {
  void sample();
}
```

ただし、メソッドにdefaultを記述することで、実装内容を記述できる

```Java
interdace A {
  default void sample(){
    System.out.println("sample");
  }
}
```
