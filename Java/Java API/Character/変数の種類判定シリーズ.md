## 変数の判定を行うメソッド群

### isAlphabetic  

受け取った文字がアルファベットであるかどうか判定する

### isDigit

受け取った文字が数字かどうかを判定する

### isLowerCase 

受け取った文字が小文字かどうかを判定する

```Java
public class Sample{

    public static void main(String[] args) {
        char a = '0';
        int num = 0;
        if (Character.isAlphabetic(a)){ // false
            num ++;
        } 
        if (Character.isDigit(a)){ //true
            num ++;
        }
        if(Character.isLowerCase(a)){ //false
            num ++;
        }
        System.out.println(num); //1
    }
}
```
