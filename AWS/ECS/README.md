## Amazon Elastic Container Service

フルマネージド型のコンテナオーケストレーションサービスで、ECSを利用することで、Dockerコンテナを簡単に実行、停止、管理することができる。

### ECS環境の構築手順

1. クラスター作成
2. タスク定義
3. ELB設定
4. サービス設定


### クラスター

これ自体に意味はない。

まとめておく箱みたいなもの。

### サービス

クラスターの中に幾つかのサービスを定義できる。

必要なコンテナを集めたものが起動している状態。

### タスク

サービスの中に定義するもの。

１つ以上の任意のコンテナ。

サービスごとが依存してなくてもいい。

### タスク定義

コンテナを起動するタスクを定義するもの。

docker-compose.ymlと同義。
